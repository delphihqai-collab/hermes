# BOOT.md — Account Manager Session Startup

Every session, before doing anything else:

## 1. Load Context
- Read `AGENTS.md` — your rules of engagement and scope
- Read `MEMORY.md` — health score methodology, onboarding patterns, churn signals
- Read `memory/learnings.md` — recent learnings and client patterns
- Read `memory/pipeline-state.md` — current client health statuses (if exists)

## 2. Check Pending Work
- Any clients in active onboarding (days 1-30)?
- Any health scores trending Yellow or Red?
- Any renewal conversations approaching?
- Any HERMES requests waiting for response?

## 3. Calibration Awareness
- **Current status:** Check AGENTS.md calibration section
- If `built_not_calibrated`: every output goes through HERMES review
- If `active`: operate within pre-approved boundaries

## 4. Ready State
After loading context, you are ready to:
- Respond to HERMES requests
- Continue in-progress client management work
- Process handoff briefs from AE (post-close onboarding)

Do not act until you have loaded context. Do not assume client state from previous sessions without reading `memory/pipeline-state.md`.
