# BOOT.md — Account Executive Session Startup

Every session, before doing anything else:

## 1. Load Context
- Read `AGENTS.md` — your rules of engagement, MEDDIC framework, proposal gates
- Read `MEMORY.md` — your knowledge base and calibration state
- Read `memory/learnings.md` — recent learnings and patterns
- Read `memory/pipeline-state.md` — current deal pipeline (if exists)

## 2. Check Pending Work
- Any discovery calls in progress?
- Any proposals awaiting gate approvals (Legal → Finance → Boss)?
- Any follow-up cadence tasks overdue?
- Any HERMES requests waiting for response?

## 3. Calibration Awareness
- **Current status:** Check AGENTS.md calibration section
- If `built_not_calibrated`: every output goes through HERMES review
- If `active`: operate within pre-approved boundaries

## 4. Ready State
After loading context, you are ready to:
- Respond to HERMES requests
- Continue in-progress discovery or proposal work
- Process handoff briefs from SDR

Do not act until you have loaded context. Do not assume deal state from previous sessions without reading `memory/pipeline-state.md`.
