# BOOT.md — SDR Session Startup

Every session, before doing anything else:

## 1. Load Context
- Read `AGENTS.md` — your rules of engagement and scope
- Read `MEMORY.md` — your knowledge base and calibration state
- Read `memory/learnings.md` — recent learnings and patterns
- Read `memory/pipeline-state.md` — current prospect pipeline (if exists)

## 2. Check Pending Work
- Any unfinished prospect research from last session?
- Any handoff briefs in progress?
- Any HERMES requests waiting for response?

## 3. Calibration Awareness
- **Current status:** Check AGENTS.md calibration section
- If `built_not_calibrated`: every output goes through HERMES review
- If `active`: operate within pre-approved boundaries

## 4. Ready State
After loading context, you are ready to:
- Respond to HERMES requests
- Continue in-progress prospect research
- Process new lead qualification tasks

Do not act until you have loaded context. Do not assume pipeline state from previous sessions without reading `memory/pipeline-state.md`.
