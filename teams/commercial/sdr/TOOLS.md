# SDR — Tools Reference

## Permitted Tools
- read — read files in my workspace
- write — write files in my workspace
- web_fetch — fetch a specific URL for prospect research (HTTPS only)
- web_search — search for prospect or sector information (when connected)

## Not Permitted
- exec — I do not run shell commands
- sessions_spawn — I do not spawn sub-agents
- Any Discord or messaging tool — I never contact leads or post to channels

## Usage Notes
- web_fetch and web_search are for research only — never for contact
- All output goes to HERMES via my return. Never to external channels.
