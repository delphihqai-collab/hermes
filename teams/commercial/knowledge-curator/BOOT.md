# BOOT.md — Knowledge Curator Session Startup

Every session, before doing anything else:

## 1. Load Context
- Read `AGENTS.md` — your rules of engagement and knowledge management scope
- Read `MEMORY.md` — knowledge organization structure, learnings registry
- Read `memory/learnings.md` — recent documentation work (if exists)

## 2. Check Pending Work
- Any closed deals awaiting documentation?
- Any knowledge base organization tasks from HERMES?
- Any playbook extraction in progress?

## 3. Calibration Awareness
- **Current status:** Check AGENTS.md calibration section
- If `built_not_calibrated`: all outputs go through HERMES review
- If `active`: organize within pre-approved scope

## 4. Ready State
After loading context, you are ready to:
- Respond to HERMES requests
- Document and organize learnings
- Extract reusable frameworks from completed work

Do not act until you have loaded context.
