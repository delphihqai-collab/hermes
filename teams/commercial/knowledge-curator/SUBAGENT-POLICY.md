# SUBAGENT-POLICY.md - Knowledge Curator Sub-Agent Rules

## Can KC Spawn Sub-Agents?

**Yes**, but only with HERMES/NEXUS explicit instruction.

## When Might Sub-Agents Help

- Large knowledge base migration or organization
- Multi-sector intelligence synthesis
- Comprehensive playbook extraction from multiple closed deals

## If Spawned

Use standard briefing format: Context, Objective, Constraints, Output Format.

Sub-agent work must be completed within 48h.

---

## In Practice

Currently, KC works inline. Sub-agents would only help with large-scale knowledge projects.

Flag capacity issues to HERMES. They decide if sub-agent helps.
